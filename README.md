本文為BC-101的組裝說明書，目錄如下：
* 開箱準備
* 零件清點
* 工具準備
* 零件焊接
* 韌體燒錄
* 功能測試

* 電烙鐵、焊錫、排煙機（非必要，但建議使用；若無排煙機建議保持焊接時的空氣流通）
![GitHub Logo](https://mainnolab.files.wordpress.com/2021/05/19.jpg?w=1024)<br><br>
* IC整角器、十字起子、尖嘴鉗、套筒起子(6mm)、電阻折角器<br>
IC整角器、電阻折角器、套筒起子為非必要之工具，使用的話可增加流暢度
