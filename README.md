# BC-101 Hookup Guide
BC-101 為基於[ Arduino Micro ](https://store.arduino.cc/usa/arduino-micro)為基礎打造而成的開源[ MIDI Controller ](https://en.wikipedia.org/wiki/MIDI_controller)<br>
本文將會簡述BC-101的組裝過程，目錄如下：
* 開箱準備
* 零件清點
* 工具準備
* 零件焊接
* 韌體燒錄
* 功能測試
